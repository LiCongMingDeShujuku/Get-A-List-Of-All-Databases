## 中文
只需转到管理Studio并输入以下语句：

```SQL
select [name] from sys.databases order by [name] asc

```
这将返回按升序排列的所有数据库的列表。


## English
Just go to management Studio and type in the following statement:

```SQL
select [name] from sys.databases order by [name] asc

```

This will return a list of ALL databases organized in ascending order.
